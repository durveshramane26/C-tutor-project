SWITCH STATEMENT


A switch statement allows a varaible to be tested for equality against a list of values.
Each value is called a case,and the variable being switched on is checked for each switch case.

                        statement(s);
                        break;

Flowchart

       The following rules apply to the switch statement:

1.The expression used in the switch statement must have an integral or enumerated type,or be of a
class type in which the class has a single conversion function to an integral or enumerated type.

2.You can have any number of case statements within switch.Each case is followed by the value to
be compared to and a semi-colon.

3.The constant expression for the case must be the4 same datatype as the variable in the switch.
and it must be a constant or literal.

4.When the variable is being switched is on is equal to the case ,the statement following that case will
execute unti a break statement is reached.

5.When a break statement is reached the switch treminates tand the flow of control jumps to the
next line following the switch statemnt.

6.Not every case need to contain a break.If no break appears,the flow of contreol will fall through
to the subsequent cases until a break is reached.

7.A switch statement can have an optional default case,which must appear tat the end of the switch.
The default case can be used for performing a task when none of the cases is true.
No break is required for the default case.
