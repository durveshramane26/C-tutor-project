UNCONDITIONAL CONTROL STATEMENT

      BREAK STATEMENT

The break statement in C programming has the following two uses:

1.When a break statement is encountered inside a loop,the loop is immediately terminated and the program control
resumes at the statement following the loop. 2.It can be used to terminate a case in the switch statement.

Flow Chart


Syntax: break; Example:
#include<stdio.h> int main() {              while(a<20)          {                      printf("value of a:%d\n",a);            a++;            if(a>15)            {           break;            }            }            return 0; } when the value of 'a' becomes greater than or equal to 15, the loop will immediately terminate and the program
control resumes at the statement following the loop.
