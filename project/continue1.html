UNCONDITIONAL CONTROL STATEMENT

      CONTINUE;

Flow Chart


The continue statement in C programming works somewhat like the break statement.Instead of forcing termination,it forces the next iteration of the loop to take place,skipping any code in between. For the 'for' loop, 'continue' statement causes the conditional test and increment portions of the loop to execute.For the while and do..while loops,continue statement causes the program control to pass to the conditional tests/ Syntax: continue; Example: #include int main() {       int a=10;       do       {           if(a==15)           {                 a=a+1;                 continue;           }           printf("value of a:%d\n",a);       }while(a<20);       return 0; }
